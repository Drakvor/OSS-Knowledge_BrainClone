# Interactive Markdown Test

This is a test document for the new interactive chunking feature.

## Section 1: Introduction

This section introduces the concept of interactive chunking for markdown documents.

Interactive chunking allows users to:
- Preview how their document will be split into chunks
- Manually edit chunk boundaries and content
- Review and approve final chunking before storage

## Section 2: Features

The interactive system provides these features:

### Preview API
- Shows recommended chunking strategy
- Displays all chunks with metadata
- No data is stored during preview

### Confirm API
- Processes user-edited chunks
- Generates embeddings
- Stores data in vector database

This document demonstrates the new interactive chunking workflow for markdown files.
